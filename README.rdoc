== MapTrack

MapTrack is a real-time tracking system.

It allows users to view multiple tracks simultaneously. In later versions it will also be
possible to inspect statistics about tracks as well as commenting and sharing them.

=== Current features

* Displaying multiple tracks on a map in (near-) real-time.
* Share tracks to other users

=== Planned features

* Retrieve statistics about a track
* Feed with current events
* Enrich tracks with location-based comments and images
